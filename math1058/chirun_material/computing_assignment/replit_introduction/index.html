Computing assignments introduction

For the computing assignments we will be using repl.it. This allows us to test the code before submission.

Initial steps

  1. Click on the "team invite" link on Blackboard.
  2. Create an account. Your real name does not need to be used, but the account name will be visible.
  3. Click on the hamburger menu, select "Teams".
  4. Select the team "MATH1058 (2021-22)"
  5. Select the assignment to work on.

Steps for an assignment

  1. Modify the ID at the top of the file. This must match either a student or iSolutions ID that the marker can match to the information on Blackboard.
  2. Check the questions on the "Markdown" tab on the right.
  3. Complete the functions in main.py.
  4. Click on the tests (the tick mark in the left bar, at the bottom).
  5. Run the tests.
  6. If needed, correct the functions so the tests pass.
  7. Press "Submit" (top right). You can submit even if the tests don't pass, but you won't get all of the marks.
