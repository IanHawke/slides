Relativistic Plasticity

  • Ian Hawke

Following on from previous reading group sessions on elasticity, we take a look at how plasticity could be formulated in relativity.

Plastic behaviour

In some materials, behaviour is

  • elastic when stress below yield $\sigma_{\text{yo}}$;
  • plastic at higher stress.
  • If loaded from unstrained to $A$ (above yield), then unloaded to $B$ (no strain), left with residual plastic stress $\epsilon^{p}$.
  • Looking for meso-scale phenomenological description.
Plasticity as described in Ottosen and Ristinmaa.

Elasticity

Recap:

  • Introduce $\mathcal{X}$, coordinates $X^A$, metric $k_{AB}$, reference space;
  • map $\mathcal{M} \leftrightarrow \mathcal{X}$ using $\psi^A_a = \nabla_a X^A$;
  • $\det(k) = n^2$; write $\Lambda = n e(k_{AB})$ through invariants of $k_{AB}$;
  • strain $\sigma_{AB} = (g_{AB} - \eta_{AB})/2$;
  • vary action for dynamics.
This quickly recaps Nils' discussion from last time. Note that I have explicitly introduced the strain tensor for later. Note also that isotropy constrains the way elasticity enters the internal energy.

Equations of motion

Dynamics:

$$ \nabla_a T^{ab} = 0, \quad T^{ab} = e u^a u^b + p^{ab}, \quad p_{ab} = 2 \frac{\partial e}{\partial g^{ab}} - e \perp_{ab}. $$

Kinematics:

$$ \begin{aligned} \nabla_{[a, b]} X^A & = 0 & \implies && \partial_t \psi^A_j + \partial_j \left( \hat{v}^k \psi^{A}_k \right) + \hat{v}^k \left( \partial_k \psi^A_j - \partial_j \psi^A_k \right) & = 0, \\ &&&& \partial_t k_{AB} + \hat{v}^k \partial_k k_{AB} & = 0. \end{aligned} $$

Here the $3+1$ split gives $u^a = \alpha^{-1} W (1, \hat{v}^i)$.

This follows GHE.

Ideal Plasticity

Assumptions:

  • Introduce yield function $f(p_{AB}) \le 0$:
    • $f < 0$ gives elastic behaviour;
    • Yield surface $f = 0$ gives "instant" plastic behaviour.
  • Plastic behaviour conserves energy and number density $n$, hence $\det(k)$.
  • No heat transport, single fluid velocity sufficient.

Consequences:

  • Dynamics unchanged;
  • reference space changes "instantly";
  • new reference space diffeomorphic to original;
  • some map, metric, describe the matter.
The key idea here is that the matter is "the same", so it makes sense that the reference spaces are diffeomorphic. The point is that the objects describing the reference space - the metric and the map - must change in some non-elastic way.

Maximum dissipation

Energy is constant, but dissipation can (must) change.

Use Helmholtz free energy $\Psi = e - sT$ to define dissipation rate $\gamma \sim \text{d} \Psi$.

Assume plasticity maximises dissipation whilst staying on yield surface $f=0$. Requires Lagrangian

$$ \mathcal{L} = -\gamma + \text{d} \lambda f. $$

Given any variable $q$, conjugate $Q = \partial_q e$, have

$$ \text{d} q = \text{d} \lambda \frac{\partial f}{\partial Q} = \text{d} \lambda \frac{\partial f}{\partial \left( \partial_q e \right)}. $$
Energy being constant is key here. We are assuming that the plasticity is driven by the microphysics and the changes happen extremely fast on scales we cannot resolve. The energy we are considering is on the meso-scale, so stays constant. This means that the action principle gives us no useful information.
However, we can use the energy functional to give us the dissipation rate. Using the laws of thermodynamics the dissipation rate is given by the change in the free energy, not the total energy, which will be linked to the change in the entropy.
For ideal plasticity we then assume that the change on the micro-scale maximises the change in the dissipation, all done whilst staying within the yield surface. We therefore write down a Lagrangian that maximises the dissipation rate whilst enforcing (through a Lagrange multiplier d lambda) that f=0.
As the dissipation rate is maximal this tells us how any state variable has to change in order to move within the yield surface. If we can identify a suitable state variable to change, this gives a return map that tells us how to modify the variables in the equations of motion.

State variables

Let $\kappa_{AB}, A^{\bar{A}, \text{el}}_j$ obey the elastic equations of motion.

$\kappa, A$ define elastic reference space $\bar{\mathcal{X}}$, diffeomorphic to $\mathcal{X}$.

Exists plastic map $P^A_{\bar{B}} \colon \bar{\mathcal{X}} \to \mathcal{X}$,

$$ \psi^A_j = P^A_{\bar{B}} A^{\bar{B}, \text{el}}_j. $$

Plastic map changes reference state:

$$ \text{d} P^A_{\bar{B}} = \text{d} \lambda \frac{\partial f}{\partial \left(\partial_{P^A_{\bar{B}}} e \right)}. $$
We know that the state of the matter is defined through some metric and some map. We can think of evolving the original metric and map using the equations of motion for elasticity. This defines a reference space. By the assumptions of perfect plasticity, this "elastic" reference space must be diffeomorphic to the true reference space. Therefore there is some map that takes us from the elastic reference space (notation uses bars) to the true one.
We start with the plastic map being the identity. We then identify the plastic map as being the state variable that we modify in the return map that keeps us on the yield surface. This leads us to the convex optimisation problem given in the final equation.

Equations of motion

Dynamics:

$$ \nabla_a T^{ab} = 0, \quad T^{ab} = e u^a u^b + p^{ab}, \quad p_{ab} = 2 \frac{\partial e}{\partial g^{ab}} - e \perp_{ab}. $$

Kinematics, map, return map:

$$ \begin{aligned} \partial_t A^{\bar{A},\text{el}}_j + \partial_j \left( \hat{v}^k A^{\bar{A},\text{el}}_k \right) + \hat{v}^k \left( \partial_k A^{\bar{A},\text{el}}_j - \partial_j A^{\bar{A},\text{el}}_k \right) & = 0, \\ \partial_t \kappa_{AB} + \hat{v}^k \partial_k \kappa_{AB} & = 0. \\ \psi^A_j & = P^A_{\bar{B}} A^{\bar{B},\text{el}}_j. \\ \text{d} P^A_{\bar{B}} & = \text{d} \lambda \frac{\partial f}{\partial \left(\partial_{P^A_{\bar{B}}} e \right)}. \end{aligned} $$
We pull everything together. We evolve stress-energy conservation, which is four equations, plus nine equations for the elastic map and nine equations for the elastic reference metric. At each stage we evaluate the yield function. If the yield function is negative we are in the elastic regime and the full map is formed using the product of the plastic and elastic maps, and the strain invariants are then computed exactly as in the elastic case.
If, however, at any point the yield function is positive then we have evolved to an unphysical state. We have to return to the yield surface. We do this using the return map, which finds the Lagrange multiplier such that changing the plastic map gets us back to the physical state on the yield surface. We can then form the full map from the plastic map and the elastic map and proceed.
It is important to note that the plastic map does not have an evolution equation. Instead, it relaxes "instantly" to enforce a physical state, assuming that this is needed.

Von Mises' yield

To complete, need EOS $e = e(k_{AB})$ and yield $f = f(p_{AB})$.

"Simplest" yield function is Von Mises' strain

$$ f = \sqrt{\tfrac{3}{2} \pi^A_B \pi^B_A} - \sigma_{\text{yield}}. $$

Can add work hardening by evolving $\sigma_{\text{yield}}$.

As $\sigma_{\text{yield}} \to \infty$ material always yields - "fluid limit".

For certain classes of metals, Von Mises' criteria has substantial experimental backing. It depends only on the anisotropic part of the pressure. This can be computed from the equation of state and the invariants of the metric. As computing the invariants involves raising indices on the reference space, this is where the spacetime metric and the full map (combining the elastic and plastic maps) comes in to the yield function.
For more complex materials there are more complex yield functions. However, we can already make life more complex by making the yield strain vary. Typically this is done by introducing a parameter (the work parameter), linking its evolution to the entropy change in the return map (that is, to the amount the material has yielded), and hence changing the point at which the body will yield in the future.

  • Ideal plasticity should extend to relativity.
